## CONTENT FUNDAMENTALS

오즈코딩스쿨 + 위디딧 톤을 융합한 **"Wake-up call 도전 + 커리어 전문성 신뢰"** 듀얼 톤. 24-29세 쉬었음 청년의 무기력감을 자극하면서, 동시에 구체적인 숫자/혜택으로 안심시키는 구조.

### 어투/인칭
- **2인칭("당신/너")보다 직설적 질문형**을 우선 사용 — "지금 당신, 무엇을 하고 있나요?" 보다 "왜 아직도 포폴이 없을까?"
- **존댓말 + 도전적 단언** 혼용 — "~합니다 / ~입니다" 베이스에 짧고 강한 단언("이것만 하세요", "끝내드립니다") 삽입
- **이모지는 거의 사용 안 함**. 강조는 색·굵기·크기로
- 한국어 본문에 영어 캡션 라벨 자주 사용 (`CURRICULUM`, `MENTORS`, `BENEFITS`)

### 카피 톤 — 3가지 모드

**1. Hook (도전적/자극적)** — Wake-up call 헤드라인
- "왜 같은 직무 같은 학력인데 누군 1억, 누군 3천만원일까?"
- "PD·BX·콘텐츠… 뭐가 다른지도 모르고 지원하면 평생 떨어진다"
- "수료증으로는 못 가는 회사들. 포폴로만 갑니다."

**2. Body (전문적/신뢰)** — 커리큘럼·혜택 설명
- "90일 동안 현업 BX 디자이너 4명에게 1:1 멘토링을 받으며, 실무 브리프 기반 포트폴리오 3종을 완성합니다."
- "수료 후 6개월간 인턴 공고를 100% 매칭해드립니다."

**3. Numbers (숫자 단언)** — 신뢰의 무기
- 모든 혜택은 숫자로 환원 — `1,300만원 → 50만원`, `90일`, `3개 포트폴리오`, `80만원 훈련장려금`, `M1 Pro`, `100% 라이브`
- 큰 숫자는 표 안에 두지 말고 **헤드라인으로** 끌어올림

### Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| "포트폴리오 3개를 손에 쥡니다" | "포폴 작성에 도움을 드립니다" |
| "PD vs BX vs 콘텐츠, 끝내드립니다" | "다양한 디자인 직무를 알려드립니다" |
| "수강료 1,300만원 → 국비로 50만원" | "국비지원이 가능합니다" |
| 영문 라벨(BENEFITS)+한글 본문 | 한글 라벨만 |
| 짧은 단언 + 단호한 마침표 | 물음표·느낌표 남발 |

### 페르소나 인용 (랜딩 곳곳에 삽입할 내부 화자)
- "쉰 지 1년… 디자인 취준은 시작도 못 했어요"

## VISUAL FOUNDATIONS

**한 줄로**: 검은 캔버스 위에 일렉트릭 블루 액센트와 큰 숫자, 표(grid) 중심 정보 밀도. AI/테크 무드. 둥근 모서리 절제. 진하고 두꺼운 헤드라인.

### 무드 보드
- **테크/AI 다크모드**: 검은 베이스(#0A0A0F) + 미세 그리드 라인(rgba(255,255,255,0.04))
- **일렉트릭 블루 1색 액센트**: `#3D5AFE` (Wedidit-inspired 변형) — CTA, 활성, 강조
- **시그널 옐로 + 레드**: 한정 수량/Wake-up 영역에만 (50만원, 마감 임박 등)
- **흰 / 회색 텍스트 4단계** — `fg-1` 순백 헤드라인, `fg-2` 본문, `fg-3` 보조, `fg-4` 캡션

### 컬러
`colors_and_type.css` 의 `--bg-1 / --bg-2 / --fg-1~4 / --accent / --accent-pressed / --signal-warn / --signal-success / --grid-line` 참조.

- **bg-1** `#0A0A0F` (페이지 베이스)
- **bg-2** `#13131A` (카드/섹션)
- **bg-3** `#1C1C26` (호버, 인풋)
- **stroke** `rgba(255,255,255,0.08)` (구분선)
- **accent** `#3D5AFE` (전기적인 블루, AI 느낌)
- **accent-hi** `#7B8CFF` (호버시 밝아짐)
- **signal-warn** `#FFD400` (긴급/한정)
- **signal-success** `#00E5A0` (수료/완료)

### 타이포그래피
- **본문**: Pretendard Variable (한·영 모두) — `fonts/PretendardVariable.woff2`
- **숫자/디스플레이**: Pretendard 900(Black) + `-letter-spacing: -0.04em`. 1억, 90일, 3개 같은 큰 숫자는 무조건 Black + tight tracking
- **English caption**: Pretendard 600 uppercase, `letter-spacing: 0.18em`
- **베이스 스케일**: 14 / 16 / 18 / 20 / 24 / 32 / 48 / 72 / 120 / 200
- **줄간격**: 본문 1.55, 헤드라인 1.1, 디스플레이 0.95

### 레이아웃 & 간격
- **데스크탑 컨텐츠 폭** 1200px. 풀블리드 섹션은 그 안에 1080px 카드
- **모바일 우선 패딩**: 24px 좌우, 섹션간 96~160px 수직 갭
- **그리드**: 12-col 또는 표 중심 (특히 가격/혜택 비교)
- **간격 스케일**: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128 / 160 (8px 베이스)

### 백그라운드
- **메인**: 단색 #0A0A0F + `0.5px` 격자 라인 8% 투명도, 80px 그리드 셀
- **히어로**: 단색 위에 라디얼 글로우 (액센트 컬러 from center) — 약하게
- **표 섹션**: bg-2 카드, 위에 그리드 라인 끔
- **금지**: 보라색-블루 그라데이션, 알록달록 배경, 텍스처 페이퍼

### 애니메이션
- **이징**: `cubic-bezier(0.22, 1, 0.36, 1)` (out-expo 변형). 220ms 베이스.
- **페이드만 사용** — 슬라이드/바운스 자제. 스크롤 인 시 8~16px만 살짝 올라오면서 fade in.
- **카운트업**: 큰 숫자는 뷰포트 진입 시 0→target 800ms 카운트업
- **금지**: 패럴랙스, 마우스 트래일, 폭죽 효과

### 호버 / 프레스
- **호버**: 액센트 컬러 → `accent-hi` (10% 밝아짐). 카드는 stroke `0.08 → 0.18`, 그림자 추가 안 함
- **프레스**: `transform: scale(0.98)`, 120ms
- **포커스**: `outline: 2px solid var(--accent); outline-offset: 2px;`

### 보더 / 코너 / 그림자
- **코너**: 작은 요소 8px, 카드 12px, 큰 모달 20px. 50%+ 라운드 금지(필 버튼 제외)
- **필 버튼**: CTA 한정. `border-radius: 9999px`
- **보더**: `1px solid rgba(255,255,255,0.08)` 기본. 강조시 `1px solid var(--accent)`
- **그림자**: 다크모드 베이스라 거의 안 씀. CTA hover 시 `0 0 0 4px rgba(61, 90, 254, 0.16)` 글로우만

### 투명도 & 블러
- **카드 배경 블러**: 사용 안 함 (다크모드 솔리드가 더 깔끔)
- **이미지 오버레이**: 멘토 사진 위에 `linear-gradient(to top, rgba(10,10,15,0.9), transparent 60%)` 로 텍스트 가독 확보

### 이미지 톤
